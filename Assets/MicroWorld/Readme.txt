Installation for Built-In Project:
 - Install the package.

Installation for URP Project:
 - Install the package.
 - In Start Screen press button “Convert To URP”.
 - Enable Opaque Texture in URP settings.

 Installation for HDRP Project:
 - Install the package.
 - In Start Screen press button “Convert To HDRP”.
 - Open inspector for any foliage material (for example MicroWorld/Resources/GrassHDRP), and press button Fix for DiffusionProfile.
